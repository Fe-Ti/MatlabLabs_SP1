ПРИМЕР РЕШЕНИЯ СЛАУ МЕТОДОМ ПРСТЫХ ИТЕРАЦИЙ

При защите лабораторной работы вам следует пояснить все строки программ, приведенных в данном примере.

Для решения задачи необходимо создать каталог в вашей папке, который будет являться рабочим каталогом MATLAB.

Первый способ приведения уравнения AX=B к виду, пригодному для решения методом простых итераций

Создадим файл, например task1.m (M – файл сценарий) 

format short
A=[5 -3 1; 
   1 -6 4; 
   1 -2 -5];
B=[1;-2;2];
 
% Задание вектора b и матрицы a
b = zeros(3,1);
a = zeros(3,3);
 
% Подготовка уравнения к методу простых итераций
for i=1:3
   b(i)=  B(i)/A(i,i);
   a(i,i)=0;
   for j=1:3
     if i~=j  
       a(i,j)=-A(i,j)/A(i,i);
     end
   end
end
 
% Проверка условия norma(a)<1
mn=norm(a)

me=norm(a,inf)
% Вывод матрицы и вектора системы: 
a
b

Второй способ приведения уравнения AX=B к виду, пригодному для решения методом простых итераций

Создадим файл task2.m (M – файл сценарий)

% Задание системы уравнений
 
format short
A=[4  3 -3; 
   1 -2 -5; 
   5 -3  1];
B=[3; 2; 1];
 
% ЧАСТЬ 2
% Подготовка СЛАУ для решения методом
% простых итераций вторым способом
 
d=[10^-3 10^-2 10^-1; 
   10^-2 10^-3 10^-2;
   2*10^-3 10^-2 3*10^-2];
 
alfa = d*A
beta = (A^(-1)-d)*B
 
% Проверка условия norma(alfa)<1
n_alfa = norm(alfa)

Cоздание М – файла функции для вычисление уравнения.

Назовем файл, например, iter.m 
function X=iter(A,B,eps)
% Решение системы уравнений AX=B 
% методом простых итераций
% A - матрица системы уравнений
% b - вектор столбец свободных членов
% norma - норма матрицы А
% eps - точность численного решения
delta=eps*(1-norm(A))/norm(A);
X0=B; % Задание начального приближения
X1=A*X0+B;
while norm(X1-X0)>delta
  X0=X1;
  X1=A*X0+B;
end;
X=X1;

Решение задачи.

Теперь в файл task1.m мы можем добавить строки: 
% Решение уравнения
format long e
X=iter(a,b,10^-5)
% Проверка
disp('Погрешность')
disp(norm(A*X-B))

% Решение уравнения
format long e
X=iter(alfa,beta,10^-5)
