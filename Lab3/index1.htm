Задание

Цели работы:

  1. Дальнейшее введение в программирование в MATLAB.
  2. Решение систем линейных алгебраических уравнений с помощью матричных операторов MATLAB.
  3. Решение систем линейных алгебраических уравнений методом итераций.

Порядок выполнения работы:

  1. Ознакомиться с заданием.
  2. Изучить теоретический раздел "Решение системы линейных алгебраических уравнений матричными средствами MATLAB".
  3. Изучить теоретический раздел "Решение системы линейных алгебраических уравнений методом простых итераций".
  4. Изучить теоретический раздел "Программирование в MATLAB (продолжение)".
  5. Создать новый рабочий каталог, например lab3.
  6. Для вашего варианта решить систему линейных алгебраических уравнений матричными средствами MATLAB. Указания:
    • Предварительно убедиться, что система уравнений невырожденная.
    • Выполнить проверку решения.
    • Решить задачу, используя четыре различных способа.
    • Результаты оформить в виде М - файла сценария, сохраненного в рабочий каталог.
  7. Для вашего варианта решить систему линейных алгебраических уравнений методом простых итераций (пример решения). Для решения задачи создать 3 М - файла (в вашем рабочем каталоге):
    • М - файл сценарий для первого способа приведения уравнения AX=B к виду, пригодному для решения методом простых итераций.
    • М - файл сценарий для второго способа приведения уравнения AX=B к виду, пригодному для решения методом простых итераций.
    • М - файл функция, реализующая алгоритм метода простых итераций.
  8. Написать отчет.
  9. Защитить отчет у преподавателя, с возможным ответом на контрольные вопросы.
